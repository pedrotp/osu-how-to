Welcome! This guide will get you started with the Dropbox HTML API

Dropbox is a great tool used by a lot of people to store and organize files in the cloud. In this guide, we will learn to communicate with the Dropbox HTTP API to directly communicate with a user's Dropbox account, and modify/add/organize their files. This can be very useful to allow users to easily upload files to our app, or to save files created in our app. As we will see later in this guide, Dropbox can also be used as a lightweight back-end storage solution for simple apps.

This is a beginner-level guide, but you are expected to have some familiarity with Javascript, HTTP Requests, HTML and CSS.
